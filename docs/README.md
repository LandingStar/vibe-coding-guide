# Documentation Map

如果你是第一次进入当前仓库，先读 `starter-surface.md`，再决定是否继续展开完整文档地图。

本目录收纳当前仓库的权威文档。

在升维后的目标下，文档应分成两层理解：

- 平台文档
  - 定义项目本体是什么、核心对象是什么、插件如何扩展
- 实例文档
  - 定义某个官方或项目自定义 workflow pack 是如何落在平台上的
